# cax-scripts
This repository contains scripts for the CARCARÁ-X beamline at SIRIUS-CNPEM.

The automated scripts execute alignment scans by controlling the mirror, DVFs,
photocollector, and slits. This procedure optimizes the luminescent
image—formed on a YAG screen—of the X-ray beam produced by the beamline's
source (a B1 dipole).

The system's primary objective is to obtain, on the YAG, an X-ray beam profile
that accurately reproduces the geometric characteristics of the electron beam,
as intended by the beamline's design.
